# Dashboard – National Health Expenditures (NHEA) 1960–2023

Este repositorio contiene el análisis y dashboard sobre el gasto nacional en salud de los Estados Unidos, basado en el dataset NHE2023 proporcionado por CMS.

# Instrucciones para ejecutar el taller

1. Descomprimir la carpeta.

2. Abrir una terminal dentro de la carpeta.

3. Ejecutar:
    pip install -r requirements.txt
    streamlit run app.py
